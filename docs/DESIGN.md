# Design System

This document outlines the core visual style and thematic properties of our design system.

## Color Palette

Our brand utilizes a specific color palette to maintain a consistent and recognizable appearance across all products and platforms.

-   **Primary Color:** `#B91C1C` - This bold, deep red is the cornerstone of our brand identity, used for key interactive elements and primary calls to action.
-   **Secondary Color:** `#111827` - A very dark grey, almost black, providing strong contrast and support for less prominent UI elements and typography.
-   **Tertiary Color:** `#4B5563` - A medium grey that serves as an accent, useful for highlights, badges, and subtle decorative touches.
-   **Neutral Color:** `#F9FAFB` - An off-white, providing a clean and bright background for our UI.

The system currently operates in a **light color mode**.

## Typography

The chosen typography ensures readability and reinforces brand personality.

-   **Headlines:** `Inter` - A modern, highly legible sans-serif font, perfect for impactful headlines.
-   **Body Text:** `Inter` - The same versatile font ensures consistent readability for all body content.
-   **Labels:** `Inter` - Consistent use for labels maintains a cohesive typographic hierarchy.

## Shape and Form

The aesthetic emphasizes:

-   **Roundedness:** A subtle roundedness (level 1) for UI elements, offering a touch of softness without sacrificing a clean, defined look.

## Spacing and Layout

-   **Spacing:** A normal (level 2) amount of spacing between UI elements, providing a balanced and comfortable visual experience.
